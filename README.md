#  My Turborepo Monorepo

Сучасний монорепозиторій на базі **Turborepo** та **pnpm**, який об'єднує фронтенд, бекенд та спільні TypeScript-типи в єдину екосистему.

##  Структура проєкту

* `apps/web` — Фронтенд-додаток на **Next.js** (працює на порту `:3000`)
* `apps/api` — Бекенд-сервіс на **NestJS** (працює на порту `:3001`)
* `packages/types` — Спільний пакет для TypeScript інтерфейсів (`@repo/types`)

##  Швидкий старт

Переконайтеся, що у вас встановлено [pnpm](https://pnpm.io/).
